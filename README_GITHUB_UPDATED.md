# 🏠 Kerala Vastu AI

> AI-Powered House Planning Assistant for Kerala Architecture

Kerala Vastu AI is a web-based intelligent house planning system that generates customized Kerala-style floor plans, construction planning reports, cost estimations, material quantity listings, construction timelines, and disaster resilience recommendations using AI.

---

## ✨ Features

### 🤖 AI-Powered Planning
- Generate customized house plans based on user requirements
- Support traditional and modern Kerala architectural styles
- Intelligent room arrangement and layout recommendations

### 🏡 Kerala Architecture Support
- Nalukettu
- Ettukettu
- Traditional Kerala Homes
- Modern Kerala Homes

### 📊 Construction Planning Reports
- Cost breakdown estimation
- Material quantity estimation
- Construction timeline generation
- Budget optimization suggestions

### 🌧️ Environmental & Risk Analysis
- Flood risk assessment
- Landslide risk recommendations
- Cyclone preparedness suggestions
- Earthquake resilience guidelines

### 📁 Export Options
- Floor Plan PNG Export
- Construction Report PDF Export
